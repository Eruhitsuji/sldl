# v1.0.0 release notes

v1.0.0 is the first stable public release of the SLDL Python compiler.

## Release focus

- Stable public documentation layout.
- English-first reference documents with Japanese companion documents.
- Official bilingual sample documents.
- Project-based build workflow.
- Release quality checks with required-file, forbidden-path, config, syntax, command, manifest, and golden snapshot checks.
- Repository-level `.gitignore` for common local and generated artifacts.

## Compatibility baseline

The public v1.0 compatibility surface is the current SLDL grammar, JSON config formats, CLI commands, official examples, and documented workflow. Historical unpublished v0 development samples are intentionally not part of the public compatibility promise.
