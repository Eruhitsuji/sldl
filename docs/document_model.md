# Document model

The v1-ready document model has three layers.

## 1. Source layer

The source layer is the `.sldl` file. It contains metadata, sections, typed content nodes, and logical relationships.

Official source examples:

- `examples/official_project_overview_en.sldl`
- `examples/official_project_overview_ja.sldl`
- `examples/research_report_en.sldl`
- `examples/research_report_ja.sldl`

## 2. Configuration layer

The configuration layer is JSON. Important configuration types are:

- `sldl.schema`: document type and rule definitions
- `sldl.project`: multi-document, multi-output build configuration
- `sldl.export_labels`: labels used by exporters
- `sldl.latex_build`: LaTeX/PDF build pipeline
- `sldl.release_check`: release-quality checks
- `sldl.snapshot_manifest`: golden output hash checks

## 3. Output layer

The output layer contains generated Markdown, HTML, LaTeX, dry-run or real PDF outputs, Mermaid-related output, JSON ASTs, and build/release manifests.

Generated files are reproducible artifacts. They are not source examples and should be regenerated by `project build`.
