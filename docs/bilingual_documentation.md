# Bilingual documentation baseline

SLDL v1.0.0 uses English as the primary reference language and provides Japanese companion documentation for practical usage.

## Policy

- English documents under `docs/` are the normative reference for public release behavior.
- Japanese documents under `docs/ja/` are companion guides intended to make the same workflow easier to use in Japanese.
- Official examples are provided in both English and Japanese.
- Public GitHub documentation should remain Markdown files stored statically under `docs/`.

## Official bilingual examples

- `examples/official_project_overview_en.sldl`
- `examples/official_project_overview_ja.sldl`
- `examples/research_report_en.sldl`
- `examples/research_report_ja.sldl`

The project overview examples explain the grammar, required JSON files, and document creation workflow in a concrete form.
